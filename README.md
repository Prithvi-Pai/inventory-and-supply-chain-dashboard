# Tableau Dashboard Project

## Overview
This project contains a Tableau dashboard and SQL script for inventory analytics.

## Files
- Tableau Workbook (.twb)
- SQL Query file (.sql)

## Problem Statement:
Organizations struggle with inefficient inventory management, delayed shipments, and lack of visibility into supply chain performance.

## Objectives :
Analyze sales, profit, and net profit margin to evaluate overall business performance. 
Monitor warehouse inventory levels and their impact on storage costs. 
Identify product departments contributing the highest order volumes and storage expenses. 
Evaluate shipment mode distribution and order fulfillment efficiency. 
Analyze geographic order patterns to understand regional demand. 
Identify products with the highest inventory investment. 
Develop an interactive Tableau dashboard that provides actionable insights for inventory and supply chain optimization.
